### 3. 수와 계산
모든 프로그래밍 언어에는 수(number) 와 계산(math)을 다루는 방법이 있습니다.

#### 수학 기호(symbol)


| symbol(기호) | 내용 |
| :-------- | :--------: | 
| + | 덧셈(Plus) | 
| - | 뺄셈(minus) | 
| / | 나눗셈(slash) | 
| * | 곱셈(asterisk) | 
| % | 나머지(percent) | 
| < | 작다(less-than) | 
| > | 크다(greater-than) | 
| <= | 작거나 같다(less-than-equal) | 
| >= | 크거나 같다(greater-than-equal)

Ex3.py
~~~Python
print("닭을 세어봅니다.")
print("암닭")
print("수닭")
print("이제 달걀도 세어봅니다.")
print(3+2+1-5+4%2-1/4+6)
print("3+2<5-7")
print(3+2<5-7)
print("3+2는 얼마죠?",3+2)
print("5-7은 얼마죠?",5-7)

print("아하 이게 False인 이유네요")
print("더 해볼까요?")
print("더 큰가요.",5>-2)
print("더 크거나 같나요?",5>=-2)
print("더 작거나 같나요?",5<=-2)
~~~
연산 우선순위는 수학에서 쓰는 순서로 괄호, 지수,곱셈과 나눗셈,덧셈과 뺄셈으로 순서를 지정합니다.
